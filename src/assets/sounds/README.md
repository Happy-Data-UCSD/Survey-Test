# Sound assets

## SND (UI sound kit)

Survey and gamified feedback use **SND01 (sine)** from [snd.dev](https://snd.dev/) via the
[`snd-lib`](https://www.npmjs.com/package/snd-lib) package. See the [SND terms of use](https://snd.dev/)
before redistributing or reusing raw audio.

## Legacy `fire-whoosh` (removed)

The former optional `fire-whoosh.mp3` for the neo-brutal “fire” pop was removed; **CELEBRATION** and
**SWIPE** from SND01 now cover that feedback in the demographic test flow.
